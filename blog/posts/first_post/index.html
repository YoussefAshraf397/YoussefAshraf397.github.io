What is Docker?

Docker is an open-source platform that enables developers to build, deploy, and manage containerized applications. Containers allow applications to run in isolated environments, ensuring consistency across different systems.

Why Use Docker?

  • Portability: Run applications consistently across various environments.
  • Efficiency: Uses fewer resources than traditional virtual machines.
  • Scalability: Easily scale applications with Docker Compose and Kubernetes.
  • Fast Deployment: Quick and reproducible application deployment.

Getting Started with Docker

  1. Install Docker: Download and install Docker from Docker’s official website.
  2. Verify Installation: Run: 
    docker --version
  3. Run a Container: Execute: 
    docker run hello-world
    This pulls and runs a basic Docker container.

Basic Docker Commands

  • List running containers: 
    docker ps
  • List all containers (including stopped ones): 
    docker ps -a
  • Stop a container: 
    docker stop <container_id>
  • Remove a container: 
    docker rm <container_id>

Creating a Simple Docker Container

Let’s create a simple Docker container running an Nginx web server.

  1. Pull the Nginx image: 
    docker pull nginx
  2. Run the container: 
    docker run -d -p 8080:80 nginx
    Now, open your browser and go to http://localhost:8080 to see Nginx running.

Building a Custom Docker Image

You can create your own Docker images using a Dockerfile. Here’s an example:

Dockerfile:

FROM ubuntu:latest
RUN apt update && apt install -y nginx
CMD ["nginx", "-g", "daemon off;"]
EXPOSE 80

Build and Run the Image:

docker build -t my-nginx .
docker run -d -p 8080:80 my-nginx

Conclusion

Docker simplifies application deployment by providing lightweight, portable, and scalable containers. It is widely used in DevOps, cloud computing, and software development.
